## 2.1 if-else

if-else的使用格式

```
    if 条件:
        满足条件时要做的事情1
        满足条件时要做的事情2
        满足条件时要做的事情3
        ...(省略)...
    else:
        不满足条件时要做的事情1
        不满足条件时要做的事情2
        不满足条件时要做的事情3
        ...(省略)...
```
demo1

```
    chePiao = 1 # 用1代表有车票，0代表没有车票
    if chePiao == 1:
        print("有车票，可以上火车")
        print("终于可以见到Ta了，美滋滋~~~")
    else:
        print("没有车票，不能上车")
        print("亲爱的，那就下次见了，一票难求啊~~~~(>_<)~~~~")
```
结果1：有车票的情况

```
    有车票，可以上火车
    终于可以见到Ta了，美滋滋~~~
```
结果2：没有车票的情况

```
    没有车票，不能上课
    亲爱的，那就下次见了，一票难求啊~~~~(>_<)~~~~
```
