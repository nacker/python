# 5.文件操作、综合应用

## 5.1.文件操作介绍
> 就是把一些存储存放起来，可以让程序下一次执行的时候直接使用，而不必重新制作一份，省时省力

## 5.2.文件的打开与关闭
1.打开文件

在python，使用open函数，可以打开一个已经存在的文件，或者
创建一个新文件

open(文件名，访问模式)

示例如下：

```
f = open('test.txt', 'w')
```

2.关闭文件

close( )

示例如下：

```
# 新建一个文件，文件名为:test.txt
    f = open('test.txt', 'w')

    # 关闭这个文件
    f.close()
```
