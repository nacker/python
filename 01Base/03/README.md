## 3.1.字符串介绍
python中字符串的格式

如下定义的变量a，存储的是数字类型的值

```
a = 100
```
如下定义的变量b，存储的是字符串类型的值

```
    b = "hello"
    或者
    b = 'hello'

```
**小总结：**
双引号或者单引号中的数据，就是字符串
